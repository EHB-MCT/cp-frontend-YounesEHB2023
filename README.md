# Up and Running
To run this project locally, follow these steps:
1. Clone repository
  ```sh
  git clone <repo-link>
```
2. Install the dependencies 

    (recommended to do on mac, windows has problems installing the packages)
  ```sh
  $ npm install
```
3. Run website
  ```sh
  $ npm run dev
```
